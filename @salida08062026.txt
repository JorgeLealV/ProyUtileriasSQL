в—Ҹ La extensiГіn mГЎs recomendada es PlantUML de jebbs (ID: jebbs.plantuml).

  CaracterГӯsticas principales:
  - Vista previa en tiempo real con Alt+D
  - Exporta a PNG, SVG, PDF, entre otros
  - Soporte para diagramas ER, secuencia, clases, etc.

  Requisito: Necesitas tener instalado Java y opcionalmente Graphviz para
  algunos tipos de diagrama. Alternativamente, puedes configurarla para usar el
  servidor de PlantUML en lГӯnea sin instalar nada local:

  En settings.json de VS Code agrega:
  "plantuml.server": "https://www.plantuml.com/plantuml",
  "plantuml.render": "PlantUMLServer"

  Esto evita instalar Java/Graphviz y funciona de inmediato con el servidor
  pГәblico.
